## 📍 PILAR 3 — Presença Online (Ser Encontrado)

> Recrutadores e clientes te encontram antes de você encontrar eles.

### GitHub
- [ ] Perfil completo com foto e bio
- [ ] GitHub Profile README (aquele que aparece na sua página)
- [ ] Pelo menos 3 repositórios públicos com README decente
- [ ] Histórico de commits consistente (verde no calendário)
- [ ] Estrelar e fazer fork de projetos que você usa — sinaliza que você é ativo

### LinkedIn
- [ ] Foto profissional (não precisa ser de terno, só boa qualidade)
- [ ] Headline: "Desenvolvedor Full Stack | JavaScript · Vue · Node"
- [ ] About section contando sua história de transição
- [ ] Listar projetos na seção "Projetos"
- [ ] Conectar com devs brasileiros e recrutadores de tech
- [ ] Postar pelo menos 1x por mês — pode ser aprendizado simples

### Presença Técnica (opcional mas diferencia)
- [ ] Dev.to ou Hashnode — escrever sobre o que está aprendendo
- [ ] Um artigo sobre algo que você achou difícil e como resolveu
> Escrever sobre o que aprendeu consolida o conhecimento E gera visibilidade.

## 📍 PILAR 4 — Preparação para o Mercado

### Para Estágio
- [ ] Currículo de 1 página focado em projetos e habilidades técnicas
- [ ] Carta de apresentação curta explicando sua transição
- [ ] Plataformas: LinkedIn, Gupy,99jobs, Programathor, GeekHunter
- [ ] Foco em empresas que valorizam portfólio mais que diploma
- [ ] Não esperar estar "pronto" — se candidatar a partir do Pilar 1 concluído

### Para CLT Jr
- [ ] Mesmo que estágio, mas com projeto âncora no ar
- [ ] Praticar desafios técnicos: HackerRank, LeetCode (nível fácil)
- [ ] Estudar as perguntas clássicas de entrevista JS
- [ ] Conhecer o básico de metodologias ágeis (Scrum, Kanban) — aparece muito

### Para Freelance
- [ ] Perfil na Workana e 99Freelas
- [ ] Primeiro job pode ser de graça ou barato — só pra ter avaliação
- [ ] Nicho sugerido: landing pages para pequenos negócios
  - Baixa complexidade técnica no início
  - Alta demanda
  - Resultado visual fácil de mostrar
- [ ] Preço inicial sugerido: R$500–R$1.500 por landing page simples
- [ ] Com o tempo: pacotes (landing + manutenção mensal)

## 📍 PILAR 5 — Soft Skills para o Contexto de Gabriel

> Este pilar é pensado especificamente para autismo nível 1 de suporte
> em ambiente profissional de tech.

### Comunicação Assíncrona (seu ambiente ideal)
- [ ] Aprender a escrever mensagens técnicas claras no Slack/Discord
- [ ] Documentar seu próprio trabalho (comentários, READMEs, PRs)
- [ ] Perguntas bem formuladas: "Tentei X, aconteceu Y, esperava Z"

### Entrevistas
- [ ] Praticar explicar projetos em voz alta (pode gravar pra si mesmo)
- [ ] Preparar 3 histórias: um desafio técnico, um erro que aprendeu, uma conquista
- [ ] Saber dizer "não sei, mas posso pesquisar" — é resposta profissional
- [ ] Empresas remotas tendem a ser mais estruturadas e previsíveis — priorizar

### Gestão de Energia no Trabalho
- [ ] Blocos de foco de 90 minutos com pausa real (não só trocar de aba)
- [ ] Definir horário de início e fim — trabalho sem limite esgota
- [ ] Ter um ritual de início de sessão (ex: ler o HISTORICO.md antes de codar)
