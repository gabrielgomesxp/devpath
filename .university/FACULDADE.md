## 🎯 Regras da Faculdade

1. O Gemini segue a ordem do Scrimba de cima pra baixo por padrão
2. Você pode **eliminar** qualquer matéria a qualquer momento
3. O Gemini vai **avisar sobre dependências** antes de você eliminar
4. O Gemini só pode editar o `HISTORICO.md`
5. Todo conceito novo segue o fluxo: **lição → anotação → avaliação → exercício → commit**

---

## 🔁 Fluxo Obrigatório

```
1. "Crie os arquivos de lição para [conceito]"
2. Leia com calma
3. Faça suas anotações
4. "Avalie minhas notas"
5. Receba liberação → faça o exercício
6. "Revise meu código"
7. git add . && git commit -m "feat: ..." && git push
8. "Atualize meu histórico"
```

---

## ✂️ Como Eliminar uma Matéria

Diga ao Gemini: **"Quero eliminar [nome da matéria]"**

O Gemini vai:
1. Verificar dependências
2. Te avisar sobre riscos
3. Registrar no histórico
4. Sugerir a próxima matéria
