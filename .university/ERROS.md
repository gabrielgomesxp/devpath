## 🔁 Protocolo do Mentor

Antes de revisar qualquer arquivo `.js`, o mentor DEVE:

1. Abrir este arquivo
2. Verificar se há padrões registrados para o conceito atual
3. Mencionar proativamente: "Lembra que você costuma [erro X]? Fique atento."
4. Após a revisão, verificar se algum erro novo se repete
5. Se sim, registrar ou incrementar o contador

### Formato de registro:

```
| 1 | Esquecer o return em funções | 3x | array_methods_01.js | 
Antes de fechar a função, pergunta: "o que ela deve devolver?" |
```
