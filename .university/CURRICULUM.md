## 📐 Detalhamento de Cada Matéria

---

### 01. Introduction
**Pode ser eliminada sem impacto.**
- Apresentação da plataforma
- Primeiro app simples
- Orientação sobre o curso

---

### 02. HTML and CSS Fundamentals
**Base adquirida. Pode ser eliminada.**
- Intro to HTML — tags, estrutura, formulários, listas
- Intro to CSS — seletores, margin, padding, flexbox
- Build a Business Card
- Build a Space Exploration Layout
- Build a Birthday Website

---

### 03. JavaScript Fundamentals ← VOCÊ ESTÁ AQUI
**Obrigatória. Núcleo de tudo.**

#### Build a Counter App
- Variáveis, funções, eventos, strings, DOM básico

#### Build a Blackjack Game
- if/else, arrays, loops, objetos, Math.random()
- Lógica de jogo completa

#### Build a Chrome Extension
- localStorage, template strings, parâmetros de função
- innerHTML, addEventListener, arrays como parâmetros

---

### 04. Tools of the Trade
**Recomendado antes de avançar.**
- Command Line Fundamentals — navegação, criar/deletar arquivos
- Command Line Power Tools — setup local
- Essential Git and GitHub Skills — branches, pull requests, merge

---

### 05. Accessible Development
**Pode ser feita em paralelo com qualquer matéria.**
- Contraste de texto e cores
- Texto alternativo e links
- Labels e radio buttons
- HTML semântico, listas, tamanho de texto
- ARIA e ARIA live regions
- Skip Navigation Link

---

### 06. Essential CSS
**Aprofundamento de CSS após o básico.**
- Build an NFT Site — semântica, hover, especificidade, botões
- Build a Portfolio — tipografia, cores, botões, detalhes visuais
- Build a Coworking Space Site — position relative/absolute, z-index

---

### 07. Essential JavaScript
**Aprofundamento de JS. Muito importante.**

#### Build a Cookie Consent
- setTimeout, element.style, formulários, preventDefault
- FormData, classList toggle, desabilitar elementos

#### Build a Meme App
- for...of, radio inputs, .includes(), classList add/remove
- querySelector, filter(), checkbox

#### Build a X (Twitter) Clone
- forEach, data attributes, event listeners
- Lógica de like/retweet, boolean flip, renderização condicional

#### Mini Projects (MUITO IMPORTANTE)
- Object Destructuring
- `.map()` Method
- `.join()` Method
- `.map()` vs `.forEach()`
- `.reduce()` Method
- Ternary Operator
- Rest Parameter
- Spread Syntax
- Short-circuiting com OR e AND
- Switch Statements
- Constructor Functions e Classes
- Debugging

---

### 08. Responsive Design
**Design que funciona em qualquer tela.**
- Unidades relativas: %, em, rem
- max-width, line-height
- Media queries e breakpoints
- Mobile-first navigation
- Flexbox image grid
- CSS Grid — span, grid lines, grid template areas, minmax
- Build a Product Page

---

### 09. APIs and Async JavaScript
**Conectar seu app ao mundo externo.**

#### Intro to APIs
- O que é uma API, clients e servers
- Primeiro fetch, Dog API, Bored API
- BoredBot completo com HTML + CSS + JS + A11y

#### URLs and REST
- HTTP Methods, endpoints, body, headers
- BlogSpace — CRUD completo com fetch
- REST API Design, recursos aninhados
- Query strings e parâmetros de URL

#### Async JavaScript
- Callbacks — como funcionam por baixo
- Promises — .then() chaining
- Projeto War — jogo de cartas com API real
- Async/Await — sintaxe moderna

#### Capstone Project
- Dashboard com Unsplash + Crypto + Clima + Tempo
- Promises em paralelo, async/await, try/catch
- Chrome extension completa

---

### 10. AI Engineering
**Diferencial enorme no mercado atual.**
- Fundamentos de AI Engineering
- Chamadas à API de LLMs (como a Anthropic/OpenAI)
- Prompt Engineering
- Temperature e exemplos
- RAG e Vector Databases (Supabase)
- AI Agents com ReAct

---

### 11. Node.js
**JavaScript no servidor.**
- Build a Node API — package.json, routing, req/res, JSON
- Parâmetros de rota, query params, modularização
- Build a Fullstack Node App — servir frontend + backend juntos
- Server-Sent Events, Event Emitter

---

### 12. Databases
**Persistência de dados.**
- SQL Queries básicas
- Criar e unir tabelas com JOIN

---

### 13. Express.js
**Framework web para Node.js.**
- Build an Express API — servidor, rotas, dados, query params, path params
- Build a Fullstack Express App — frontend + banco + dropdown + busca
- Authentication — registro, login, hash de senha, express-session, JWT, carrinho

---

### 14. User Interface Design
**Design profissional de interfaces.**
- UI Design Fundamentals — hierarquia, contraste, tipografia, espaçamento
- Desafios progressivos de design
- Design a Simple Layout
- Refactor a Metrics Dashboard — cor, contraste, tipografia, espaçamento, media queries

---

### 15. React.js Fundamentals
**A biblioteca frontend mais usada no mercado.**

#### Static Pages
- Primeiro código React, JSX, componentes, props
- ReactFacts Project completo

#### Data-Driven React
- Travel Journal — props, .map(), arrays de componentes

#### React State
- Chef Claude — useState, formulários, renderização condicional
- Sound Pads — estado compartilhado, estilos dinâmicos
- Integração com AI (receitas com LLM)

#### Side Effects
- Meme Generator — useEffect, fetch, dependencies array
- Componentes controlados

#### Capstone 1 — Tenzies (jogo de dados)
#### Capstone 2 — Assembly Endgame (jogo da forca)

---

### 16. Testing
**Garantir que seu código funciona.**
- Primeiros testes com Vitest/Jest
- Testar interações do usuário
- Mock de serviços externos
- Testes de acessibilidade + TDD

---

### 17. Advanced React.js
**React profissional.**
- Reusabilidade — componentes e hooks avançados
- Performance — memo, useMemo, useCallback
- Routing — React Router avançado
- Persistence — localStorage com React
- Authentication — fluxo completo de autenticação

---

### 18. TypeScript
**JavaScript com tipos.**

#### TypeScript Fundamentals
- Pizza App — tipos básicos, tipos customizados, arrays tipados
- Union types, literal types, type narrowing
- Utility Types, Omit, Generics

#### TypeScript em React
- Tipando useState, props, componentes, funções
- Assembly Endgame com TypeScript

#### Solo Project — Typed Tenzies

---

### 19. Next.js
**O framework React para produção.**
- File-based routing, páginas, layouts
- Otimização de imagens e links
- Dynamic Routes, nested routes
- Rendering Strategies — SSG, SSR, Client Components
- Forms nativos e Next Form Component
- Categorias, busca, páginas dinâmicas
