# 📖 Lições

Esta pasta contém os arquivos de lição criados pelo mentor
antes de cada exercício novo.

Nunca pule direto para o exercício sem ler as lições primeiro.

---

## Como funciona

Para cada conceito novo, o mentor cria uma subpasta com até 4 arquivos:

```
lessons/
  [conceito]/
    01-o-problema.md          ← leia primeiro
    02-a-solucao.md           ← leia segundo
    03-como-usar.md           ← leia terceiro (tem sugestão de pesquisa)
    04-exercicios-progressivos.js  ← só abrir após avaliação das notas
```

---

## Fluxo obrigatório

```
1. Ler 01, 02 e 03 com calma
2. Escrever minhas anotações
3. Pedir avaliação ao mentor
4. Receber nota ≥ 7 → ir para o 04
```
