# 🚀 Projetos

Esta pasta contém projetos que eu idealizei e construí do zero.

Diferente dos exercícios (que seguem o currículo),
projetos partem de uma ideia minha ou de uma sugestão do mentor
como desafio integrador ao final de uma etapa.

---

## Dois tipos de projeto

**Sugerido pelo mentor** — ao concluir uma etapa do currículo,
o mentor propõe um projeto integrando os conceitos aprendidos.

**Inventado por mim** — quando vejo algo que quero construir
e decido simplesmente começar.

---

## Estrutura de cada projeto

```
projects/
  nome-do-projeto/
    index.html
    style.css
    main.js
    README.md     ← obrigatório: o que é, tecnologias, o que aprendi
```

O `README.md` dentro de cada projeto é o que vai para o portfólio.
